# API-project

## App using api: 
  FBI API from: https://www.fbi.gov/wanted/api

## App have features like:
 - get random criminal
 
 - get youngest criminal
 
 - get oldest criminal

 - get result list searching by hair color
 
 - export all criminals to xlsx
 
 - get all criminals to MySql database

## Used framework:
 - Hibernate
 
 - Swing

## libraries:
- google gson
 
- lombok
 
- poi
